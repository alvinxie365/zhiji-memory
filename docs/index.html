📦 安装

Installation

1
Clone the repository
git clone https://github.com/alvinxie365/zhiji-memory.git
cd zhiji-memory
2
Run the install script
bash install.sh
3
Enable in config.yaml
memory:
  provider: zhiji_memory

重启 Hermes Agent 后自动生效 | Restart Hermes Agent to activate

✦ ✦ ✦

⚙️ 配置

Configuration

启用插件 / Enable the Plugin
Add to ~/.hermes/config.yaml
memory:
provider: zhiji_memory
LLM 炼化（可选）/ LLM Refinement (Optional)
Supports any OpenAI-compatible API. If not configured, runs in rule-only mode.
ZHIJI_LLM_API_KEY=sk-... # 专用 API Key (最高优先级)
ZHIJI_LLM_BASE_URL=... # API 端点
ZHIJI_LLM_MODEL=gpt-4o # 模型名称
每日备份 / Daily Backup (Recommended)
Add to crontab — runs at 2am daily
0 2 * * * bash ~/.hermes/scripts/zhiji-daily-backup.sh
